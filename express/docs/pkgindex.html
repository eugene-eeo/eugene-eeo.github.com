Lightweight library for designing web APIs, based on Flask and Werkzeug for acting as high-level routers.

Companion to the Synrails library- first you have your TCP servers, now you have an entire web API.

A simple “Hello World!” API:

import express
app = express()

@app.serve()
def serve(request):
    method = request.form["method"].strip()
    return app.Model._run(string=method)

@app.event("hello")
def hello_world():
    return "Hello World!"

app.listen(5000)

Run that, and start another Python REPL, preferrably with requests installed:

>>> call = {"method":"hello"}
>>> requests.post("http://localhost:5000/", data=call).text
'{"value":"Hello World!","status":"True"}'

Note

The express API is not a regular Python namespace module but is an Express instance, i.e.:

import express
app = express()

However, you can still do imports via the express module:

import express.contrib
import express.helpers
