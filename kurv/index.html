Simple tool to sign files using Ed25519 + Blake2b, with support for keyrings. Uses monocypher under the hood.

Quickstart

  1. Generate keypair: 
    $ kurv -g id
$ ls
... id.pub id.priv
  2. Sign: 
    $ kurv -sP id.priv file.txt > file-signed.txt
$ cat file-signed.txt
...
----BEGIN KURV SIGNATURE----
2XPeY4lCTUOLBKdt7tfMPDAYuhaD...
----END KURV SIGNATURE----
  3. Check: 
    $ kurv -cp id.pub file-signed.txt
$ # or
$ cat file-signed.txt | kurv -cp id.pub
    Keyring support using $KURV_KEYRING: 
    $ export KURV_KEYRING='...'
$ kurv -c file-signed.txt
