Simple set of tools for signing / encryption, built on Monocypher.

Signing

  • support for pipes
  • produces armoured signatures
  • can sign arbitrarily long messages by signing blake2b(M)
  • keyring support
# generate keypair
$ kurv -g id
$ cat id.{pub,priv}
P/Vkey2lTatqY5dHpxgxPrdSnbv4cQih+Gn7Flgs1tQ=
...

# signing
$ kurv -sk id.priv README
...
----BEGIN KURV SIGNATURE----
bQ3Mx3Vb0tdmb1GF2f0jgLm8GJpvQhYFYe2kkrOHkno5
yOhnAi2Dl9Gmpt/Tx9aG4VyWTUjfdVyx7QI1xrpgCQ==
----END KURV SIGNATURE----

# checking signatures
$ kurv -ck id.pub signed-file && echo "ok"
ok

Encryption

  • no proof of sender
  • encrypt arbitrarily long messages
  • password / public key encryption
  • see Lock Stream Format
$ echo "Hello" \
   | luck -ek id.pk \  # encrypt for recepient
   | luck -dk id.sk    # decrypt
Hello
$ echo "World" \
   | luck -ep "password" \ # with password
   | luck -dp "password"
World
